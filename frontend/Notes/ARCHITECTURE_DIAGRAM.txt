KEY DIFFERENCES:

  Feature              │ Admin CMS           │ Frontend
  ─────────────────────┼─────────────────────┼─────────────────────
  Data Source          │ Supabase (live)     │ Local JSON (cache)
  Update Type          │ Real-time           │ When "Update Local" clicked
  Network Calls        │ Every page load     │ None (zero calls)
  Speed                │ Depends on network  │ Instant (local files)
  Freshness            │ Always current      │ Controlled by user
  Use Case             │ Content Management  │ Public browsing
  Offline Support      │ No (needs network)  │ Yes (works offline)
